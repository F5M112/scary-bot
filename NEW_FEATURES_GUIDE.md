# 🆕 ميزتان جديدتان

## 1️⃣ 🔴 Kick Live Alerts

### الإعداد:
1. ادخل **Dashboard → Kick Live**
2. اختر سيرفرك
3. اضغط **"+ إضافة قناة"**
4. اكتب اسم قناة Kick (مثل: `D7miMax`) → اضغط 🔍 بحث
5. ستظهر معلومات القناة مع الصورة والمتابعين
6. اختر:
   - **قناة التنبيه في ديسكورد**
   - **منشن** (اختياري): @everyone أو رول معين
   - **نص الرسالة** مع متغيرات
7. اضغط **إضافة**

### المتغيرات المتاحة في الرسالة:
- `{streamer}` — اسم القناة
- `{title}` — عنوان البث
- `{link}` — رابط القناة
- `{viewers}` — عدد المشاهدين

### كيف يعمل:
- البوت يفحص كل **دقيقتين** حالة القنوات المسجلة
- عندما تبدأ قناة بثاً → يرسل تنبيهاً مع Embed كامل (صورة، عنوان، فئة)
- لن يكرر التنبيه لنفس البث
- الحد الأقصى: **10 قنوات** لكل سيرفر

## 2️⃣ 📿 الأذكار التلقائية

### الإعداد:
1. ادخل **Dashboard → الأذكار**
2. اختر سيرفرك
3. **فعّل** المفتاح في الأعلى
4. اختر **قناة الإرسال**
5. اختر **الفترة الزمنية** (15 دقيقة → يومياً)
6. اختر **الفئات**:
   - 🌅 أذكار الصباح
   - 🌙 أذكار المساء
   - 📿 أذكار عامة
   - 🤲 أدعية
   - 🌹 الصلاة على النبي ﷺ
   - 📖 آيات قرآنية
7. (اختياري) خصص العنوان واللون
8. اضغط **حفظ**
9. للاختبار اضغط **إرسال تجريبي**

### المحتوى المتوفر:
- **40+** ذكر ودعاء وآية موزعة على 6 فئات
- المحتوى من أصول إسلامية موثوقة (الأذكار النبوية، الصحيحين، القرآن)
- البوت يختار عشوائياً ويتجنب التكرار المباشر

### مثال إعداد مقترح:
- ⏱️ كل ساعة
- 📿 أذكار عامة + 🤲 أدعية + 🌹 صلاة على النبي
- النتيجة: ذكر مختلف كل ساعة على مدار اليوم

## ⚠️ ملاحظات مهمة

### للميزتين:
- يجب أن يكون البوت **في السيرفر** ولديه صلاحية الإرسال في القناة المختارة
- يجب أن يكون الحساب **بباقة نشطة** (classic أو premium)
- إذا تم تعطيل الحساب أو السيرفر من الإدارة → كل التنبيهات تتوقف فوراً

### الخدمات تعمل في الخلفية:
عند تشغيل الباك إند ستظهر:
```
✅ تم الاتصال بقاعدة البيانات
✅ البوت الرئيسي جاهز
🟢 خدمة Kick Live monitor بدأت (كل دقيقتين)
🕌 خدمة الأذكار التلقائية بدأت
🚀 السيرفر يعمل على المنفذ 5000
```

## 🚀 خطوات التطبيق

1. **استخرج الملفات الجديدة من الـ ZIP**
2. **احتفظ بـ `.env` القديم**
3. أعد تشغيل الباك إند:
   ```bash
   npm install   # لو فيه مكتبات جديدة
   npm run dev
   ```
4. أعد تشغيل الفرونت إند:
   ```bash
   npm run dev
   ```
5. افتح Dashboard وستجد التبويبين الجديدين في الـ Sidebar 🎉
